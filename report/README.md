# Project Report

This directory contains the final report for the Bayesian Life Expectancy Modeling project.

## Contents

### final_report.pdf

Comprehensive analysis covering:

* Bayesian regression modeling
* Prior predictive checks
* Posterior predictive checks
* MCMC diagnostics
* WAIC model comparison
* Leave-One-Out Cross Validation (LOO-CV)
* Bayes Factors
* Life expectancy prediction
* Development status analysis
* Regional child mortality analysis

The report documents the full methodology, model assumptions, validation procedures, and key findings.
